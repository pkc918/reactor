## Installation

```bash
pnpm dlx shadcn-vue@latest add spinner
# or: npx shadcn-vue@latest add spinner
# or: yarn dlx shadcn-vue@latest add spinner
# or: bunx --bun shadcn-vue@latest add spinner
```

## Usage

```vue
<script setup lang="ts">
import { Spinner } from '@/components/ui/Spinner'
</script>

<template>
  <Spinner />
</template>
```
