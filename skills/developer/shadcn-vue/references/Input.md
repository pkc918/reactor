## Installation

```bash
pnpm dlx shadcn-vue@latest add input
# or: npx shadcn-vue@latest add input
# or: yarn dlx shadcn-vue@latest add input
# or: bunx --bun shadcn-vue@latest add input
```

## Usage

```vue
<script setup lang="ts">
import { Input } from '@/components/ui/input'
</script>

<template>
  <Input />
</template>
```
