## Installation

```bash
pnpm dlx shadcn-vue@latest add kbd
# or: npx shadcn-vue@latest add kbd
# or: yarn dlx shadcn-vue@latest add kbd
# or: bunx --bun shadcn-vue@latest add kbd
```

## Usage

```vue
<script setup lang="ts">
import { Kbd } from '@/components/ui/kbd'
</script>

<template>
  <Kbd>Ctrl</Kbd>
</template>
```
